Управление Topics

Создавайте, просматривайте и удаляйте Kafka topic-и. Topic -- это основная единица организации данных в Kafka: producer-ы пишут в topic-и, consumer-ы читают из них.

Что такое Topic?

Topic -- категория, в которую публикуются записи. Topic-и разделены на partition-ы для параллелизма и масштабируемости. Каждый partition -- упорядоченная, неизменяемая последовательность записей. Replication factor определяет, сколько копий каждого partition-а хранится на разных broker-ах для отказоустойчивости.
