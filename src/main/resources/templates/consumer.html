Consumer

Наблюдайте за сообщениями из Kafka topic-ов в реальном времени через Server-Sent Events (SSE). Лента обновляется автоматически при поступлении новых сообщений.

Типы Consumer-ов в Kafka

Simple Consumer: слушает topic-и с auto-commit. Прост в использовании, но может привести к at-most-once доставке.
Manual Ack Consumer: коммитит offset вручную после обработки. Обеспечивает at-least-once гарантию.
Batch Consumer: обрабатывает сообщения пачками для повышения throughput.
Filtered Consumer: применяет фильтрацию, обрабатывая только подходящие сообщения.
Consumer с обработкой ошибок: использует retry и Dead Letter Topic (DLT) для обработки сбоев.
